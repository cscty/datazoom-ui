# Avatar 头像

Avatar 组件用来表示人物或对象，支持图片或者文字作为 Avatar。

# 基础用法

使用 shape 和 size 属性设置 Avatar 的形状和大小。

circle
square

# Attributes

参数说明类型可选值默认值
size尺寸string / numberlarge / default / smalldefault
shape形状stringcircle / squarecircle
srcAvatar 图片资源地址string——
altAvatar 图片原生 alt 属性string——
fit当展示类型为图片时，设置图片如何适应容器stringcover / fit / contain / none / scale-downcover
